Coinbase OAuth2 Integration

Why OAuth2?

OAuth2 is the secure, recommended method for Coinbase API integration. It provides better security than API keys and is the standard for production applications.

How It Works

  1. Click "Authenticate with Coinbase OAuth2"
  2. You'll be redirected to Coinbase's secure login page
  3. Log in with your Coinbase credentials
  4. Authorize Arbion to access your account data
  5. You'll be redirected back to Arbion with secure tokens

Permissions Granted

  • Read your account information and balances
  • View transaction history
  • Access wallet data for portfolio analysis
  • No trading permissions (read-only access)

Security Features

  • Secure state parameter validation
  • Tokens are encrypted and stored securely
  • Automatic token refresh when expired
  • No sensitive credentials stored in browser

Important Notes

  • This requires a valid Coinbase account
  • Tokens expire after 2 hours and will be refreshed automatically
  • You can revoke access anytime from your Coinbase settings
  • Legacy API key method is deprecated
