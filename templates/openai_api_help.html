OpenAI API Key Setup

Complete guide to setting up your OpenAI API key with proper permissions

Common Error: Missing Permissions

Error 401: "You have insufficient permissions for this operation. Missing scopes: model.request"

This means your API key doesn't have the required permissions to use OpenAI's models.

Step 1: Create a New API Key

1. Go to OpenAI API Keys

2. Click "Create new secret key"

3. Give it a name (e.g., "Arbion Trading Bot")

4. Select permissions: All or ensure "model.request" is included

5. Click "Create secret key"

6. Copy the key immediately (it won't be shown again)

Step 2: Check Your Organization Role

1. Go to Organization Settings

2. Ensure your role is Owner or Writer

3. If you're a Reader, ask an Owner to upgrade your permissions

Step 3: Verify Your Project Access

1. Go to Projects

2. Select your project or create a new one

3. Ensure you're an Owner or Member

4. Create the API key within this project

Step 4: Check Your Billing

1. Go to Usage & Billing

2. Add a payment method if you haven't already

3. Ensure you have available credit or billing limits

4. Free trial accounts have limited access

Important Notes

  • • API keys created before recent updates may lack required scopes
  • • Always use the latest API key format from platform.openai.com
  • • Keep your API key secure and never share it publicly
  • • The trading bot uses minimal tokens for testing connections
